# ⏱️ Pencatat Waktu Kejadian

Aplikasi web sederhana untuk merekam waktu kejadian pribadi (misal: waktu kencing, minum, olahraga, dll) beserta jeda antar kejadian.

## 🎯 Fitur

- ⏱️ Rekam waktu **tepat saat tombol ditekan** (jam:menit:detik)
- ⏳ Hitung otomatis **jeda antar kejadian** (berapa lama Anda tahan)
- 📊 Statistik: total catatan, catatan hari ini, rata-rata jeda
- 💾 Data tersimpan **lokal di browser** (localStorage) — privasi terjaga
- 📤 Ekspor data ke JSON untuk backup
- 📱 Responsif — bisa di-install ke Home Screen HP

## 🚀 Deploy ke Vercel

1. Fork / clone repo ini
2. Login ke [vercel.com](https://vercel.com)
3. Klik **Add New → Project**
4. Import repository ini
5. Klik **Deploy** — selesai!

## 🛠️ Teknologi

- HTML + CSS + Vanilla JavaScript
- Tidak ada framework, tidak ada build step
- Data disimpan via `localStorage` (100% di sisi klien)
